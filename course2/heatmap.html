3. Heatmaps

A heatmap is a figure that uses colour rather than numbers to highlight value differences in a table.

The figure below shows the relative global temperatures over the last 150 years or so and is taken from my article Topical Plots: Global Warming Heatmaps .

One of the easiest ways of creating a heatmap is with the Seaborn library. You can see from the code below that Seaborn simply take a Pandas dataframe as a parameter and displays the appropiate map as a Matplotlib chart. Run the code to see the heatmap.

utils.codeElement("heatmap1", codefile="./code/heatmap1.py")

You can achieve a similar chart using the imshow() function in Matplotlib.
