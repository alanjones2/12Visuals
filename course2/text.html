1. Simple text

Sometimes, a graphic isn't necessary, or even the best option to communicate data. When only a couple values are to be presented, simple text is fine and may even be better than a graph. Let's take an example.

The weather in London, UK, seems to getting hotter in the Summer. The maximum temperature in July 2022 was 27.2 degrees Celsius, which is quite hot for the UK. In 2012 it was 24.2 degrees.

Now, look at the bar graph - it shows a temperature change from one year to another. And while it's clear that the temperature went up a few degrees, you can't clearly see exactly how much or precisely what those temperatures are.

So, let's see how some text only visuals can give us a better idea of what is going on.

The code below is a Mathplotlib chart but with no figure plotted in it - we simply position text in the right places using ax.text(), where the first two parameters are the x/y coordinates. We turn of the axes, ticks etc. with the statement ax3.axis('off'), so that we end up with just the text.

Run the code below to see the figure that we have created and feel free to change some of the code to achieve a different effect (maybe the font size, colors or x/y coordinates). (Remember, to run the code click the green play button at the bottom right.)

import matplotlib.pyplot as plt #Set up variables years = ['2012','2022'] temps = [24.2,27.2] caption = f"The maximum temperature in July 2022 was {temps[1]}째C" caption2 = f"That's {temps[1]-temps[0]}째 up from 2012" fig2, ax2 = plt.subplots(figsize=(5,1)) ax2.text(0, 0.9, temps[1], verticalalignment='top', horizontalalignment='left', color='red', fontsize=18, fontweight = 'bold') ax2.text(0.2, 0.9, caption, verticalalignment='top', horizontalalignment='left', color='Black', fontsize=10) ax2.text(0.2, 0.55, caption2, verticalalignment='top', horizontalalignment='left', color='darkgrey', fontsize=6) ax2.axis('off') fig2
Here is a different version which is a bit bolder and more eye-catching than the previous one.
import matplotlib.pyplot as plt #Set up variables years = ['2012','2022'] temps = [24.2,27.2] caption = f"The maximum temperature in July 2022 was {temps[1]}째C" caption2 = f"That's {temps[1]-temps[0]}째 up from 2012" fig3, ax3 = plt.subplots(figsize=(3,2)) ax3.text(0.5, 0.9, temps[1], verticalalignment='top', horizontalalignment='center', color='darkblue', fontsize=64, fontweight = 'bold') ax3.text(0.5, 0.3, caption, verticalalignment='top', horizontalalignment='center', color='blue', fontsize=8) ax3.text(0.5, 0.2, caption2, verticalalignment='top', horizontalalignment='center', color='lightblue', fontsize=6) ax3.axis('off') fig3
