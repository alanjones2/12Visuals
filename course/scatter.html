4. Scatterplot

Scatterplots are used to show the relationship between two variables. The example below uses a data file that records weather data for each month in London in 2018. This data set records the maximum, minimum and mean temperatures (Tmax, Tmin and Tmean), the rainfall (Rain) and the number of hours of sunshine (Sun) for each month.

It looks something like this:

We'll create a scatter plot that plots the level of rainfall in millimetres, against the number of hours of sunshine. Of course, when it is raining the sun is not normally shining, so you would expect to see fewer hours of sunshine when it rains more. You will notice that the scatter diagram clearly shows this relationship.

The code below uses the Matplotlib to create the chart.

import pandas as pd import matplotlib.pyplot as plt from pyodide.http import open_url url='https://raw.githubusercontent.com/alanjones2/12Visuals/main/data/london2018-2.csv' url_content = open_url(url) weather = pd.read_csv(url_content) fig, ax = plt.subplots() weather.plot.scatter(x='Rain', y='Sun', ax=ax) fig

When you have run this code, clear the output and try changing the y parameter to 'Tmean' - that will plot rainfall against the mean temperature Tmean. Run it again and you will see that, as you would expect, the temperature is higher when it is not raining. (If you don't Clear Output) before running the code again, the new plot will be appended to the old one.)
