Learn by doing: an interactive course on how to create the 12 most useful graphs and charts in Python.

In this course you will learn how to create the 12 most useful charts by running the code in a live interactive Python window. Learn by doing not just watching and reading. You can change the code or write your own and run it live in the browser.

Storytelling with Data by Cole Nussbaumer Knaflic is a comprehensive book about data visualization in business. It's author has taught data visualization at Google over several years and this book is based on her extensive hands-on exoerience.

One of the first things you learn in the book is that the author relies on only 12 different types of visualization. The book describes these visuals and their use but does not go into implementation.

That's what we will do here. In this course we will implement the 12 visuals in Python and you can run, and edit, all of the code in the course.

The course is based upon the Medium articles (no paywall):

  • 12 Essential Visualizations and How to Implement Them - Part 1
  • 12 Essential Visualizations and How to Implement Them - Part 2

where you can find more detailed explanations of the 12 implementations for Matplotlib and Streamlit (this course deals only with Python libraries that will run in PyScript - i.e. not Streamlit). You can read them first or just get going with the first part of the course by clicking 'Next' at the bottom of the page.

How to use this course

Each of the code examples below can be run by hovering over the code window and clicking the green play button. The result of the code will be shown in the box. Any errors will also be displayed. You can edit the code and re-run it and clear the ouput with the button.

I in some cases the code in previous windows must be run before the subsequent code. So, if you get an error that says that some variable has not been defined, you probably just need to run the earlier code.

The code being run here is PyScript. This is a version of Python that runs in a browser. It's mostly just standard Python but there are some differences, for example, to display a Matplotlib chart, you just need to write is name. In a standard Python environment you would need to write plt.show().
